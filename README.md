# overview

- `api/`
  - node.js
  - express
  - graphql

- `frontend/`
  - react
  - webpack
  - react-apollo

# instructions

1. navigate to `api/`
2. run `npm install` or `yarn`
3. run `npm run watch` or `yarn watch`
4. navigate to `client/`
5. run `npm install` or `yarn`
6. run `npm run watch` or `yarn watch`
7. access `http://localhost:8080` in your browser
